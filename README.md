## Komentorivi toiminnot

### Testaus

Testit suoritetaan komennolla 

- mvn test

Testikattavuusraportti luodaan komennolla

- mvn jacoco:report

### Checkstyle

- mvn jxr:jxr checkstyle:checkstyle

### Jar

Jarin generointi komennolla

- mvn package
